# 家系図ジェネレータ

## 概要

シンプルなテキストファイルから家系図を生成するジェネレータです。

## テキストの書式について

- 列が2つのタブ区切りテキストファイル(TSV)です。
- その他の列は有っても無くても構いません。
- A列、B列とします。
- それぞれ親のフルネーム、子供の名前を格納します。
- A列、B列の何れか一つに値が入ります。両方に値があるのは書式エラーです。
- A列に値のある行を親行、B列に値のある行を子行とします。
- A列の先頭が # の行はコメント行として無視されます。

### フルネーム書式

- 氏と名をスペースを開けずに記述します。
- 氏はカギ括弧で括ります。
  - `[磯野]カツオ`
- 氏は前後どちらも受け付けます。
  - `ジョナサン[ジョースター]`
  - `Michael Joseph[Jackson]`
- 一人目の親の氏名からその家名、氏の前後位置が定まり、子供のフルネームへ適用されます。
- 名に `*` を含めるとそれ以降を非表示にできます。これを用いて同姓同名の別人を扱えます。
